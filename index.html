Você lembra ou já ouviou fala do o famoso “jogo da cobrinha”. Nele, o jogador assume forma de uma cobra e deve se alimentar dos itens dispostos pela tela para aumentar o tamanho de sua personagem.

Vale dizer que o funcionamento do jogo é muito similar ao do título em que é inspirado. Caixinhas estão dispostos ao longo de todo o mapa, e é com eles que o jogador conquista massa aumentar o tamanho da cobrinha. Como no tradicional Snake, é necessário tomar cuidado para não colidir com o próprio corpo, pois, caso contrário, sua cobrinha morrerá.

SN@K3 G@ME

INSTRUÇÕES:

Utilize as teclas de direção para orientar os movimentos da Snake:

( → ) Seta para direita (right)
( ← ) Seta para esquerda (left)
( ↑ ) Seta para cima (up)
( ↓ ) Seta para baixo (down)

Caso a Snake saia do mapa é só apertar qualquer uma das teclas de direção para ela voltar para o mapa.

Quando finalizar o jogo, "Game Over" aperte F5 ou atalizar a página para iniciar um novo jogo.
